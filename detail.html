作品について

説明： HELLO VIETNAM
作品URL： https://tienlong98.github.io/HelloVietNam
制作期間： 1ヶ月
作業環境： Adobe XD ・ Visual Studio Code
使用言語： HTML・SCSS・Java Script

制作意図

  • ひとつサイトを作ってみたい
  • もっと多くの人にベトナムを知ってほしい

今回このサイトは1年生のコンテストとして制作しました。新型コロナウイルスのパンデミックが終わった後、旅行したい人たちが増えていきます。 山に行くのか海に行くのか。食べ物や費用も大切なポイントです。そんな中で、 ベトナム旅行は魅力的な選択肢です。この作品は、ベトナムを観光する上で知っておくと良い情報を文化、料理、観光スポットの面から写真付きで紹介します。 誰もが素晴らしい旅行をすることができるように、ベトナムの良いところをまとめ上げました。
